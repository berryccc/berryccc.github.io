日行者App产品需求文档

Posted on 2019-03-23


一、产品介绍

日行者APP旨在提供在日留学生更便利的生活服务，帮助用户快速获取资讯，例如租房、手机办理、WiFi办理等商家信息并同时提供交流社区的内容服务平台。

三、全局说明

3.1 功能权限

  1. 登录状态下可以进行所有操作
  2. 未登录状态不可以进行消息，我的，社区的关注、评论以及点赞。要进行上述操作需要跳转到登录页面进行登录。

    3.2 页面内交互

    3.2.1 弹窗

  • 确认框：居中显示（1）关闭弹窗，返回原页面；（2）前往指定页面
  • 底部选择弹窗：从底部向上滑出（1）选项；（2）取消键，点击关闭或者点击页面其他区域关闭，关闭向下滑动关闭

3.2.2 Toast

  • Toast提示：居中显示，显示时间1s，1s后结束

3.2.3 加载刷新

  • 进入页面时的加载状态
  • 页面下拉刷新：顺序从1到3，顶部位置

3.3 页面异常

从左到右页面的对应的异常情况如下：

  • 无数据页面
  • 加载失败页面
  • 网络异常页面，点击刷新时，页面重新加载

四、详细功能说明

4.1 登录/注册页

4.1.1 登录

  1. 手机号输入框，获得焦点提示字符消失，向上滑动调出数字键盘
  2. 密码输入框
  3. 登录按钮，当手机号和密码输入完全时可用。异常提示：
    • 未填写手机号或密码：请填写手机号和密码
    • 填写的手机尚未注册：手机号尚未注册
    • 手机号已注册但密码错误：密码错误
    • 网络问题：请检查网络连接

4.1.2 注册

点击注册标签进入注册页面，点击输入框获得焦点，提示字符消失，向上滑动调出键盘

  1. 手机输入框，支持数字输入，需要选择对应的地区码
  2. 获取验证码，点击后60s内不可用，倒计时结束后变为可用，按钮文字变为“获取验证码”
  3. 密码输入框，支持输入6-20位字符，支持输入字母、数字以及字符
  4. 输入昵称，昵称可为英文、中文或数字

4.1.3 重置密码

  1. 获得焦点提示字符消失，向上滑动调出数字键盘
  2. 60s内不可用，倒计时结束后变为可用，获取验证码按钮
  3. 获取焦点时，提示字符消失，向上滑动调出密码键盘
  4. 验证码非空且密码输入完成（满足6~20位密码要求）时，确定按钮显示可用
  5. 异常提示：
    • 验证码错误：验证码填写错误
    • 密码超过字数限制，系统自动删除：超过字数限制
    • 网络异常：请检查网络连接

4.2 社区

4.2.1 社区动态

  1. 点击全部，下方滑出标签模块，显示所有标签，同时只能选择一个标签，选中标签后，标签高亮显示，标签选择模块上滑消失，同时社区里的内容切换到包含此标签的动态
  2. 跳转到我的关注页面，具体看4.2.2
  3. 发布动态icon，点击跳转至动态发布页
  4. 关注按钮，可关注该动态的用户
  5. 点击赞，爱心高亮并后方的数字加1，再次点击高亮消失，且后方数字减1
  6. 点击跳转至动态评论页
  7. 用户头像，点击可查看该用户的个人主页
  8. 点击动态内容区域以及空白区域可跳转到动态详情页

4.2.2 我的关注

  • 无关注人时，显示无内容的空页面
  • 有关注人时，已关注的用户未发表任何动态，也显示无内容的空页面
  • 有关注人时，已关注用户发表的动态按是时间顺序排，发布时间较近的排在前面
  • 每个动态显示用户的头像、昵称、标签以及对应的内容

4.2.3 发布动态

  1. 动态内容区及添加照片，点击添加照片，打开系统相册，需要开启访问系统相册权限，未开启日行者APP读取照片的权限时，弹出提示框，内容显示“未开通读取照片的权限，是否开通”；照片最多可添加9张
  2. 多选分类标签可同时选择一至三个
    单选分类标签只可选择一个，选中标签高亮显示
  3. 点击发布，动态内容将上传到个人主页同时展示在社区首页

4.2.4 动态详情

  • 从社区点击用户动态进入动态详情页面，可查看动态详情内容、评论以及评论回复
  • 点击底部输入框，向上滑出键盘，最多可写140个字符，点击页面空白区域可隐藏键盘

4.3 服务

4.3.1 服务首页

  1. 进入服务页后，下方的服务标签高亮处理上方6个分类，点击相应的服务分类，进入服务列表
  2. 推荐服务，由后台配置推荐服务

4.3.2 服务详情

  • 进入服务详情，查看服务内容，包括轮播图、详情、价格，以及推荐套餐，推荐套餐由后台配置
  • 轮播图5s一张向右进行切换
  • 点击收藏将此商家加入我的收藏
  • 点击立刻咨询跳转至聊天框并自动发送信息至客服

    4.4 消息

    4.4.1 系统消息

  • 消息列表，如有未读的消息，右方显示小红点，点击进入消息详情页并退出之后小红点消失
  • 消息列表第一位默认为系统消息，进入系统消息页面显示所有的未读新消息，点击新消息之后切换到相关的页面，点击关注人，则进入对方的个人主页，点击评论或者点赞则进入该动态的详情页

    4.4.2 聊天消息

  • 点击私信消息，进入聊天界面
  • 点击聊天输入框右边的语音按钮可以切换至语音输入
  • 点击输入框右侧的笑脸进入表情，点击加号则显示发送照片或者拍摄
  • 聊天设置：
    • 选择置顶聊天，则在聊天列表中系统消息的下方按照置顶时间的先后顺序依次排列。取消置顶都将不在对该联系人优先排列，默认消息时间先后排列。
    • 选择屏蔽消息，则该联系人所发送的消息不做提醒，只在聊天列表中显示红点。取消后正常消息提醒。
    • 点亮加入黑名单，则不再接受到该联系人的信息，对方发送信息时将会收到提示。取消后不收到之前未发送的信息，新的信息正常接收。

      4.5 我的

      4.5.1 个人主页

  1. 我的页面，可以查看关注以及粉丝数，点击进去可看具体的用户
  2. 点击头像进入个人信息设置
  3. 个人主页，照片墙显示我发布的动态中所有的照片，动态即用户发送的内容和图片，其他用户可以查看“我的”主页

    4.5.2 个人信息

点击我的页面的头像进入个人设置，可对个人的信息进行修改，包括头像、昵称、签名等。

4.5.3 其他功能

  • 我的收藏，展示收藏的服务列表，顶部有四个服务分类标签
  • 关于我们，显示版本号等信息，在该页面可以退出登录
